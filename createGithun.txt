1. github에서 생성한 파일
2. 로컬에서 저장함!




cd c:/study/repository/git_test1
git add .
-> 현재 풀더에서 변경된 파일을 커밋 대상으로 등록한다 .은 현재 폴더의 모든사항

git commit -m "입력할것"
->변경 내용을 하나의 기록(커밋)으로 저장합니다. 
-m 뒤에는 이번 작업을 설명하는 메시지를 작성합니다.

git push origin main
지금까지 만든 커밋을 GitHub 저장소로 업로드합니다.
origin은 원격 저장소(보통 GitHub)를 뜻합니다.
main은 브랜치 이름입니다.
